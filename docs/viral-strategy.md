# OrchestraAI -- Viral Strategy

## Positioning

**One-liner**: "The open-source AI marketing platform that gets smarter with every campaign."

**Target audience**: Solo founders, indie hackers, small marketing teams, growth engineers.

**Why now**: Single-platform tools are blind to cross-platform performance. No open-source solution orchestrates AI agents across 9 platforms with guardrailed bidding.

## Growth Loops

### Loop 1: Data Flywheel
```
More campaigns -> More engagement data -> Better embeddings -> Better recommendations -> Better campaigns
```
Each tenant's data makes the system smarter. More platforms connected = richer signal = better ROI.

### Loop 2: Open Source Network Effect
```
GitHub stars -> Contributors -> More connectors -> More users -> More stars
```
Apache 2.0 license enables community-driven platform connectors. Each new connector increases the platform's value for all users.

### Loop 3: Content Virality
```
User publishes optimized content -> Higher engagement -> User shares tool -> New users
```
OrchestraAI-optimized content performs measurably better, creating organic word-of-mouth.
